# 🐺 Loberia Shield | Tactical XDR

**Loberia Shield** is a lightweight, high-performance security engine built to protect endpoints and cloud infrastructure. Developed by **Loberia Tactical Group**, it combines local heuristic analysis with global cloud intelligence.

## 🛡️ Key Features
* **Cloud-Native Scanning:** Real-time file reputation checks via VirusTotal API (70+ Antivirus engines).
* **Multi-Platform Support:** Unified core for Windows (Workstations) and Linux (VPN/Servers).
* **Persistence Engine:** Anti-kill mechanisms to ensure the shield remains active.
* **Automated Quarantine:** Isolated environment for detected threats.
